<product_requirements>
The Skylio Hub is a travel aggregator portal featuring comprehensive, multi-language travel guides.

**Core User Requirements:**
1.  **Bilingual Support:** The application must be fully functional in Spanish and English, including all UI and guide content. Hebrew support was explicitly removed.
2.  **Content-Rich Guides:** All guides, especially the Must-See Destinations, must feature engaging, narrative descriptions with consistent detail across both languages. They must also include links to Wikipedia and popular, short, informative YouTube videos.
3.  **AI-Powered Travel Assistant:** The app must include a smart travel assistant.
    *   **Integration:** The assistant is integrated into a dedicated AI Assistant tab within the Travel Guides modal.
    *   **Functionality:** It should provide detailed, tailored travel plans (hotels, airlines, activities, etc.) for any destination worldwide. It must intelligently differentiate between general queries (e.g., Paris) and specific requests (e.g., romantic tours in Paris).
    *   **Technology:** The integration must use a free API that does not require a credit card (Groq was selected and implemented).
    *   **Bilingual Output:** The AI's response language must match the user's selected language in the app.
</product_requirements>
